#### 🚀 项目简介
提供一套功能完备、专业可靠的心理测评系统源码，基于主流技术栈【SpringBoot + Mybatis-Plus + Vue + MySQL + Redis】构建，采用前后端分离架构，代码规范清晰、易于扩展与二次开发。无论您是心理学研究者、教育机构、企业HR部门，还是软件开发者，本项目均可作为高效、稳定的基础平台，助您快速搭建专属的在线心理测评服务。

#### 🎯 核心功能
量表丰富：内置 260+ 专业测评量表，涵盖常用心理评估方向，支持自定义添加与配置。
管理便捷：支持量表管理、用户分组（如学生、员工）、权限控制、测评记录查询与统计分析。
报告生成：自动生成结构化测评报告，支持 Word / PDF 导出，内容专业清晰。
题型多样：支持单选、多选、矩阵题等多种题型，可自定义计分规则与测评流程。
数据看板：通过可视化图表展示群体趋势与统计结果，辅助决策分析。

#### 💼 专业与可靠
测评流程设计严谨，符合心理学测量规范，保障数据有效性与信度。
系统架构清晰，前后端分离，接口权限控制完善，数据定期备份。
代码结构规范、注释完整，模块解耦，便于定制化开发与功能扩展。
